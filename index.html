Resumo

Nesse documento, abordaremos os seguintes tópicos sobre o sistema de controle de estoque

  • Introdução;
  • Front-End;
  • Respositório do projeto;
  • Desenvolvedores;

Este artigo descreve o desenvolvimento de um sistema de controle de estoque utilizando as tecnologias JavaScript, HTML e CSS.

O sistema foi desenvolvido em duas etapas:

  • a primeira etapa consistiu no desenvolvimento da interface web;
  • a segunda etapa consistiu no desenvolvimento do back-end.

Desenvolvimento da interface web

A interface web foi desenvolvida utilizando JavaScript, HTML e CSS. O JavaScript foi utilizado para implementar a lógica do sistema, o HTML foi utilizado para definir a estrutura da página, e o CSS foi utilizado para definir o estilo da página.

A interface web foi dividida em duas partes: uma parte pública, que é acessível a todos os usuários, e uma parte privada, que é acessível apenas a usuários autorizados. A parte pública permite que os usuários visualizem o estoque de produtos, e a parte privada permite que os usuários realizem operações de entrada, saída e alteração de estoque.

Desenvolvimento Back-end

O back-end foi desenvolvido utilizando C#, Entity Framework e banco local. O C# foi utilizado para implementar a lógica do back-end, o Entity Framework foi utilizado para gerenciar o banco de dados, e o banco local foi utilizado para armazenar os dados do sistema.

O back-end foi dividido em duas partes: uma parte de serviços, que é responsável por fornecer serviços para a interface web, e uma parte de dados, que é responsável por gerenciar o banco de dados.
